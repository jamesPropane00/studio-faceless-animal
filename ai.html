Recommended Models

Open models that actually work

These are models tested with OpenCode that fit on consumer hardware. Pull any of them with Ollama and start building.

Q
Qwen3-Coder

Alibaba's code-specialized model. Strong tool calling, long context, fits on 24GB GPUs. The current best open coding model for agentic use.

  • ollama pull qwen3-coder
  • 128K context window
  • Strong function calling
D
DeepSeek Coder V3

DeepSeek's code model. Competitive with closed models on coding benchmarks. Great for refactoring, debugging, and generating full files.

  • ollama pull deepseek-coder
  • Strong at long-form codegen
  • MIT license
G
Gemma 4

Google's open model. Smaller footprint, fast inference, solid all-around performance for chat, coding, and tool-based tasks.

  • ollama pull gemma4
  • Runs on 16GB RAM
  • Great for daily agent use
How It Connects

The faceless AI workflow

One terminal. Two tools. No cloud. Here is how the pieces fit together.

1
Install Ollama

Download from ollama.com. It runs as a local server on port 11434. Pull any model with one command.

2
Install OpenCode

`npm install -g opencode-ai` or use the install script. It connects to Ollama's API automatically with a config file.

3
Configure & Run

Point OpenCode to Ollama's endpoint, select your model, and start coding. The agent edits files, runs commands, and searches your codebase.
