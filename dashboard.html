Recovery

✓ Recovery Code On File

Your recovery code was saved when you created your account. For security, only a verification hash is stored — the original raw code is not held anywhere on the platform and cannot be displayed again. If you have lost your code, use the button below to generate and record a new one.

Last set: —

⚠ Recovery Setup Required

Your account does not have a recovery code on file. Without one, you cannot recover access if your password is lost. Generate a new recovery code now and store it somewhere private and safe — a password manager or a written note kept offline.

⚡ New Recovery Code — Save This Now

This is the only time this code will ever be visible. As soon as you leave this screen, the raw code is gone — only a secure hash remains on the platform. Copy it immediately and store it in a private, safe location.

⚠ Navigate away and this code is unrecoverable. Save it before continuing.

Critical: This platform does not store your email address or real-world identity. Your password and recovery code are the only proofs of account ownership that exist. If both are lost, account restoration is not possible — by design, and without exception.

Access Policy

How your account works on Signal

Your account is fully platform-native. There is no email address, no social login, and no external account dependency. Your username, Signal ID, password, and recovery code are the only credentials that belong to you here.

Signal ID

Your Signal ID is a unique platform identifier separate from your username. Share it with members you trust — it will power contact-style connections in a future build. The platform does not expose it publicly on your behalf.

What stays private

Your role, plan tier, account status, recovery state, and storage data shown here are visible only to you while signed in. Your recovery code is never transmitted in plaintext, never stored in raw form, and is not viewable by anyone — including admins.
